# 🚀 Space Shooter Game

A simple space-themed shooter game developed in Python. Control your spaceship, destroy enemy ships, and survive as long as possible while achieving the highest score.

## 📖 Description

Space Shooter is a 2D arcade-style game where players pilot a spaceship through space, avoiding enemy attacks and shooting down hostile ships. The game becomes progressively more challenging as the player's score increases.

## ✨ Features

- Smooth spaceship movement
- Bullet shooting system
- Enemy spaceship spawning
- Score tracking system
- Simple and intuitive controls
- Space and shooting sound effects

## 🛠️ Technologies Used

- Python 
- Pygame
